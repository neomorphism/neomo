# Index

## Common Information

> Padding and Margin

```
m-{0-10}    // margin
ml-{0-10}   // margin-left
mr-{0-10}   // margin-right
mt-{0-10}   // margin-top
mb-{0-10}   // margin-bottom

p-{0-10}    // padding
pl-{0-10}   // padding-left
pr-{0-10}   // padding-right
pt-{0-10}   // padding-top
pb-{0-10}   // padding-bottom
```

### Example

```html
<div class="pl-1 pr-1">Some code</div>
```
