# Helpers

## Colored Links Information

> Colored Links usage

```html
<!DOCTYPE html>
<html charset="utf-8">
  <head>
    <title>test</title>
    <link rel="stylesheet" type="text/css" href="/css/neomo.css" />
  </head>

  <body>
    <div class="column size-8 m-3">
      <a
        href="https://getbootstrap.com/docs/5.0/helpers/colored-links/"
        class="link-info"
        >Info Link</a
      >
    </div>
    <div class="column size-8 m-3">
      <a
        href="https://bulma.io/documentation/helpers/color-helpers/"
        class="link-success"
        >Success Link</a
      >
    </div>
    <div class="column size-8 m-3">
      <a
        href="https://startbootstrap.com/previews/sb-admin-2"
        class="link-danger"
        >Danger Link</a
      >
    </div>
    <div class="column size-8 m-3">
      <a
        href="https://demo.themesberg.com/neumorphism-ui/html/components/forms.html"
        class="link-warning"
        >Warning Link</a
      >
    </div>
  </body>
</html>
```

## Typography Information

> Typography usage

```html

<html charset="utf-8">

<head>
  <title>test</title>
  <link rel="stylesheet" type="text/css" href="/css/neomo.css" />
```

> text-center usage in inline classes

```html
<!DOCTYPE html>
<html charset="utf-8">

<head>
  <title>test</title>
  <link rel="stylesheet" type="text/css" href="/css/neomo.css" />
</head>

<body>
  <div class="text-center neomo">
    <h1>Regarding an inline class, write "text-center" in its parent.</h1>
    <span class="font-size-1">This is font-size-1</span> </br>
    <span class="font-size-2">This is font-size-2</span> </br>
    <span class="font-size-3">This is font-size-3</span> </br>
    <span class="font-size-4">This is font-size-4</span> </br>
    <span class="font-size-5">This is font-size-5</span> </br>
    <span class="font-size-6">This is font-size-6</span> </br>
  </div>
</body>

</html>
```

> text-center usage in block classes

```html
<!DOCTYPE html>
<html charset="utf-8">

<head>
  <title>test</title>
  <link rel="stylesheet" type="text/css" href="/css/neomo.css" />
</head>

<body>
  <div class="dark">
    <h1 class="text-center">Regarding a block class, write "text-center" in it.</h1> </br>
    <p class="font-size-1 text-center">This is font-size-1</p>
    <p class="font-size-2 text-center">This is font-size-2</p>
    <p class="font-size-3 text-center">This is font-size-3</p>
    <p class="font-size-4 text-center">This is font-size-4</p>
    <p class="font-size-5 text-center">This is font-size-5</p>
    <p class="font-size-6 text-center">This is font-size-6</p>
  </div>
</body>

</html>
```
